Описание
Проект имеет 2 раздела. Test_ui с тестами для сайта SwagLabs (https://www.saucedemo.com/) и Test_git_api с тестирование репозиториев в GitHub.

В test_ui находятся тесты проверяющие: 
  1) test_SwagLabs.py содержит smoke тесты:
    1.1)  test_full_correct – Заходит на сайт, производит сортировку товаров по цене, выбирает первый товар, доходит до успешного завершения покупки.
    1.2) test_login_correct – Проверка корректного логина.
    1.3) test_login_incorrect – Проверка с неверным логином.
  2) test_interface содержит тесты сортировки товаров и уникальных изображений карточек товаров:
    2.1) test_filtress_AZ – Фильтр карточек товаров в алфавитном порядке
    2.2) test_filtress_ZA – Фильтр карточек товаров в обратном алфавитном порядке
    2.3) test_filtress_price_low – Фильтр карточек товара по возрастанию цены
    2.4) test_filtress_price_high – Фильтр карточек товара по убыванию цены
    2.5) test_all_images_unique_correct – Проверка что все изображения в карточках товаров уникальные
    2.6) test_all_images_unique_incorrect - Проверка что все изображения в карточках товаров уникальные, выдает ошибку 
  3) test_price содержит тесты связанные с ценами.
    3.1) test_price_equal – тест единой цены товара на разных этапах покупки
    3.2) test_price_summ – проверка верности итоговой суммы к оплате за несколько товаров

В test_git_api тестируется работа с репозиториями в GitHub:
  	1) test_get_repo проверяет доступ к репозиториям и выдает список этих репозиториев.
  	2) test_new_repo создает новый репозиторий и  выдает список существующих репозиториев.
  	3) test_delete_repo удаляет созданный репозиторий и выдает список существующих репозиториев.

Инструкция
Для test_ui:
  1) Скачать репозиторий на компьютер
  2) Установить содержимое файла requirements.txt
  3) Файл .env содержит: тестовый сайт; логины и пароли тестового сайта 
  4) Тесты имеют маркеры:
    -	smoke: Тесты в test_SwagLabs
    -	correct:  Тесты в test_interface отрабатывающие верно  
    -	incorrect: Тесты в test_interface отрабатывающие с ошибкой
    -	price: Тесты в test_price 


Для test_git_api:
  1) Скачать репозиторий на компьютер
  2) В файл test_git_api/.env прописать:
  	- Ваш токен от GitHub с правами для создания и удаления репозитория
    - Имя пользователя GitHub владеющего токеном
    - Уникальное имя репозитория
  3) Запустить тесты можно с маркером git
